# Cómo utilizar tu archivo de Twitter

La forma más sencilla de usar tu archivo de datos de Twitter es a través de la interfaz de navegación del archivo proporcionada en este fichero. Basta con hacer doble clic en `index.html` en el directorio raíz y podrás navegar por la historia de Tweets desde el interior de tu navegador.

---

En la carpeta de `datos`, tu archivo de Twitter está presente en dos formatos: JSON y CSV exportados por mes y año.

* CSV es un formato genérico que puede ser importado por muchas herramientas de datos, hojas de cálculo o utilizado usando simplemente un lenguaje de programación.

## JSON para Desarrolladores

* La exportación JSON contiene una representación completa de tus Tweets tal como son devueltos por la v1.1 de la API de Twitter. Mira https://dev.twitter.com/docs/api/1.1 para más información.
* La exportación JSON también se utiliza para alimentar la interfaz de navegación del archivo (index.html).
* Para utilizar la exportación en un analizador JSON genérico en cualquier lenguaje, elimina la primera y última línea de cada archivo.

Para realizar comentarios, hacer preguntas o compartir ideas con otros desarrolladores de Twitter, únete a los foros de discusión en https://dev.twitter.com.
